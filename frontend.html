Definión.

El frontend se refiere a la parte del desarrollo web que se ocupa de la interfaz de usuario y la experiencia visual de un sitio o aplicaación web.

Fundamentos del frontend.

1.- HTML

Es el lenguaje de marcado que estructura el contenido de las páginas web, HTML define los elementos básicos de una página, como títulos, párrafos, imágenes, enlaces, tablas, etc.

2.- CSS

Lenguaje que se utiliza para describir la presentación de un documento escrito en HTML. CSS permite definir colores, fuentes, margenes,alineación, diseño de la disposición de la página.

3.- JAVASCRIPT

Lenguaje que permite agregar interactividad a las páginas web, se pueden hacer cosas como responder a clics del usuario, cambiar el contenido, realizar peticiones a servidores sin recargar la página.
